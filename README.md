This project is a Machine Learning-based Thyroid Disease Prediction System that predicts whether a person may have a thyroid disorder based on clinical, demographic, and biochemical features. It also provides a Streamlit Web UI for easy interaction.

üìå Features Used

The model uses the following features:

Demographic & Medical History

age: Age of the patient

sex: Gender of the patient

on_thyroxine: Whether the patient is on thyroxine medication

query_on_thyroxine: Query if the patient was on thyroxine

on_antithyroid_meds: Whether the patient is on antithyroid medication

sick: Whether the patient is currently sick

pregnant: Pregnancy status

thyroid_surgery: Any past thyroid surgery

I131_treatment: Whether the patient had I-131 treatment

query_hypothyroid: Query if patient is hypothyroid

query_hyperthyroid: Query if patient is hyperthyroid

lithium: Whether patient is on lithium treatment

goitre: Presence of goitre

tumor: Presence of tumor

hypopituitary: Hypopituitary status

psych: Presence of psychological condition

Biochemical Lab Tests

TSH: Thyroid Stimulating Hormone level

T3: Triiodothyronine level

TT4: Total Thyroxine level

T4U: Thyroxine Uptake level

FTI: Free Thyroxine Index
